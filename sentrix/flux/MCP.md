# MCP Servers — flux

## fetch
- **Command:** `mcp-server-fetch`
- **Transport:** stdio
- **What it does:** Fetches the content of any public URL and returns it as text. flux uses this to research UI component patterns, look up accessibility guidelines (WCAG), reference design system documentation (Material, Radix, shadcn/ui), and evaluate implementation feasibility when briefing pixel.

## mermaid-mcp
- **Command:** `mermaid-mcp`
- **Transport:** stdio
- **What it does:** Renders Mermaid diagram syntax into SVG/PNG diagrams. flux uses this to generate component hierarchy diagrams, layout flow diagrams, and state diagrams for variant states — artefacts included in canvas submissions and component library entries.
